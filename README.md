# Global-Sales-Report-Dashboard

##  Project Overview

This project is a **Global Sales Dashboard** created using Power BI.
It analyzes a store dataset with branches across different regions and markets.
The dashboard helps stakeholders understand **sales performance, shipping patterns, and regional trends**.

---

##  Data Cleaning

The dataset was cleaned before visualization:

* Removed missing/null values
* Corrected data types (Date, Sales, Quantity)
* Removed duplicate records
* Ensured consistency in columns

---

##  Features Implemented

###  1. Segmentation (Filters)

Interactive slicers are used for:

* Country
* Region
* Market

 Users can filter and explore data dynamically.

---

###  2. Shipping Analysis

* A **Pie Chart** is used to show
   Percentage of shipping based on **Ship Mode**
* Displays:

  * Standard Class
  * Second Class
  * First Class
  * Same Day

---

###  3. Sales Analysis (Bar Charts)

The following bar charts are created:

*  Sales by City
*  Sales by State
*  Sales by Region
*  Sales by Market

All charts use **Sum of Sales** for accurate comparison.

---

###  4. Tables (Detailed View)

Tables are created for:

* City-wise sales
* State-wise sales
* Region-wise sales
* Market-wise sales

 Helps users view exact values along with visuals.

---

###  5. KPI Cards

Key metrics displayed at the top:

*  Total Profit: 286.40K
*  Total Sales: 2.30M
*  Total Orders: 9994

---

###  6. Map Visualization

* A map is used to show **Sales by City location**
* Helps identify geographical distribution of sales

---

##  Key Insights

* Standard Class shipping is used the most
* Western region shows higher sales performance
* Certain cities and states contribute more to total sales
* Market-wise sales are dominated by the US
